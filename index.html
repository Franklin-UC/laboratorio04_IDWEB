Pasos para hacer café

  1. Hervir agua
  2. Agregar café molido al filtro
  3. Verter agua caliente sobre el café
  4. Esperar 3-4 minutos
  5. Servir en taza

Algoritmo: Encontrar el número mayor en una lista

  1. Inicializar variable "mayor" con el primer número de la lista
  2. Para cada número en la lista (repetición):
    • Si el número actual es mayor que "mayor" (condición)
    • Entonces actualizar "mayor" con el número actual
  3. Continuar hasta revisar todos los números
  4. Retornar el valor de "mayor"
